# Vue-Flask

Quick web Dashboard / REST API



![preview](docs/preview.png)


## Stack

* Backend
    * Flask
    * Gunicorn

* Frontend
    * VueJS
    * Bootstrap


## Use

### Requirements

* npm
* python3
* virtualenv


#### Setup

    make
    
#### Testing
    
    
    make check
   
    
#### Gunicorn foreground
    
    
    make run
